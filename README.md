# e02-github
## Exercise 2: GitHub Basics

### Summary

This is a starter exercise to get everyone up to speed in using GitHub for
computational assignments.

### Instructions

Please use Atom or another text editor to revise the accompanying `bio.md` in
the following steps. Don't do everything at once: the point is to see how
successive revisions are logged and tracked.

1. Replace the placeholder string `?fullname?` with your full name (first and
last names) and the string `?prefname?` and your preferred first name (what
you'd like to be called in class). Please be sure to delete the question marks:
they're just part of the placeholders.

1. Stage that change and then commit it to your local repository with a short
log message.

1. Replace the letters `?coding?` with a few words or a sentence or two about
any background you have in coding.

1. Handle the remaining sections in a similar way. There are three: one on your
economics background, one on your statistics background, and one on the kinds
of skills you're hoping to get out of the class.

1. Stage and commit that change with an appropriate log message.

### Submitting

Once you're happy with your bio and have committed all of the changes to
your local repository, please push the changes to GitHub. At that point, you're
done: you have submitted your answer.
